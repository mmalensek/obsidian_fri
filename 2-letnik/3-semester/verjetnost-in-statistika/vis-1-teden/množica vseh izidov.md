[[osnovni-pojmi-verjetnosti]];
$$\Omega = \{w_{1}, w_2, ..., w_n\}$$
je [[množica vseh izidov]] (ozr. rezultatov) nekega <font color="#92cddc">slučajnega poskusa,</font> kjer so $w_1, w_2, ..., w_n$ <font color="#92cddc">elementarni</font> izidi in so vsi <font color="#92cddc">enako</font> verjetni.
